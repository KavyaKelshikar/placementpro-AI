# PlacementPro AI

PlacementPro AI is a production-ready campus placement portal built with React, Tailwind CSS, Express, MongoDB, and Python services.

## Structure

- Frontend: React + Vite + Tailwind
- Backend: Express + JWT + MongoDB
- Python: resume analysis, skill matching, recommendation engine

## Run locally

### Frontend

```bash
npm install
npm run dev
```

### Backend

```bash
cd backend
npm install
npm run dev
```

### Python services

```bash
cd python
pip install -r requirements.txt
```
